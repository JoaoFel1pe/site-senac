O que são chaves criptográficas?

As chaves criptográficas são elementos essenciais em sistemas de criptografia. Elas são utilizadas para garantir a confidencialidade, integridade e autenticidade dos dados transmitidos e armazenados.


Tipos de chaves criptográficas

  • Chave simétrica
  • Chave assimétrica
  • Hash

Chave simétrica

A chave simétrica, também conhecida como chave de criptografia convencional, utiliza a mesma chave para criptografar e descriptografar os dados. Ambas as partes envolvidas na comunicação precisam compartilhar a mesma chave de forma segura.


Chave assimétrica

A chave assimétrica, também chamada de chave pública/privada, utiliza um par de chaves: uma pública e outra privada. A chave pública é compartilhada amplamente, enquanto a chave privada é mantida em segredo. Elas são utilizadas para criptografar e descriptografar os dados de forma segura.


Hash

O hash é um valor resultante de um algoritmo de hash aplicado a um conjunto de dados. Ele é usado para verificar a integridade dos dados e garantir que eles não tenham sido modificados durante a transmissão ou armazenamento.


Como a ProYou usa as chaves?

Funcionário

Quando um funcionário se cadastra no site, ele recebe uma chave única. Essa chave está associada ao ID da empresa. Com essa chave, o funcionário pode enviar reclamações de forma anônima para o RH da empresa.

Empresa

A empresa possui um ID único. Através desse ID, ela consegue visualizar quais chaves estão associadas a ela e identificar as reclamações enviadas pelos funcionários.

Fluxo do Sistema

  1. O funcionário se cadastra no ProYou.
  2. Após o cadastro, o funcionário recebe uma chave única.
  3. O funcionário utiliza a chave para enviar uma reclamação para o RH da empresa de forma anônima.
  4. A empresa utiliza seu ID para visualizar as chaves associadas a ela.
  5. A empresa pode acessar as reclamações enviadas pelos funcionários utilizando as chaves correspondentes.
